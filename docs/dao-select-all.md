# select-all
## 使用方法
### HTML
```HTML
<input type="text" v-model="" [dao-select-all]>
<textarea [dao-select-all]></textarea>
<div [dao-select-all]></div>
```

## 属性
| 参数名            | 类型   | 说明                                       | 默认值  | 是否必填 |
| :------------- | ---- | ---------------------------------------- | ---- | ---- |
| dao-select-all | -    | 使用时只需加上该属性即可，不用赋值。作用标签可以是 input、textarea、div。 | -    | 是    |
